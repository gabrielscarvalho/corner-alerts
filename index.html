Explanations

Mode of use


Import required files. 
				<script src="node_modules/jquery/dist/jquery.min.js"></script>
				<script src="lib/js/corner-alerts.js"></script>
				<link rel="stylesheet" href="lib/css/style.css"/>
Initiate the plugin 
				$(document).ready(function(){ 
					$('body').cornerAlertsInit(options);
				});
You can edit the options replacing the following json: 
	
			{
			'top-left': {
				'id': 'corner-alert-top-left', //unique id to place
				'top': '20px',  
				'left': '20px',
				'right': 'unset',
				'bottom': 'unset',
				'addOrder': 'bottom', // when adds a new data, will put in the bottom (as last)
				'extra-class': ''
			},
			'top-right': {
				'id': 'corner-alert-top-right',
				'top': '20px',
				'left': 'unset',
				'right': '20px',
				'bottom': 'unset',
				'addOrder': 'bottom',
				'extra-class': ''
			},
			'bottom-left': {
				'id': 'corner-alert-bottom-left',
				'top': 'unset',
				'left': '20px',
				'right': 'unset',
				'bottom': '20px',
				'addOrder': 'top',
				'extra-class': ''
			},
			'bottom-right': {
				'id': 'corner-alert-bottom-right',
				'top': 'unset',
				'left': 'unset',
				'right': '20px',
				'bottom': '20px',
				'addOrder': 'top',  // when adds a new data, will put in the top (as first)
				'extra-class': ''
			}

		};
It means that you can create a new node to put anywhere in the page. After this, you just need to run: 

			    //                     page-locale, action
				$('#test2').cornerAlert('bottom-right', 'show');
				$('#test2').cornerAlert('bottom-right', 'hide');
